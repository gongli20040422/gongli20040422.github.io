如何使用hexo + github 搭建个人博客

前言

最近很多流行使用github+hexo搭建博客，因为完全免费、稳定。本文介绍如何搭建博客。

简介

什么是github？

官网 ：

简单来说，GitHub是一个代码托管云服务网站，帮助开发者存储和管理其项目源代码，且能够追踪、记录并控制用户对其代码的修改。
甚至你可以简单粗暴的把它当作一个网盘，用来存储任何东西。

什么是hexo？

官网：

Hexo 是一个快速、简洁且高效的静态博客框架，它基于 Node.js 运行，可以将我们撰写的 Markdown 文档解析渲染成静态的 HTML 网页。

环境搭建

下载并安装 node.js 和 git ;没什么好说的，默认下一步就好。
node.js 官网：

Git官网:

安装完后检查一下, win + R 打开cmd输入 node -v、npm -v 和 git --version

1.创建github账号。

2.连接github。

2.1桌面右键 – 选择“Git bash Here“。

2.2创建ssh密匙：

输入ssh-keygen -t rsa -C “GitHub 邮箱”然后一路回车。

2.3添加密匙。

进入 C:\Users\用户名.ssh\id_rsa.pub 并且复制里面的内容。

打开Github，找到GitHub_Settings_keys。

GitHub_Settings_keys URL：https://github.com/settings/keys

点击New SSH Key

Title 随便起一个，key框里面填刚刚复制的内容。添加完后的电机 Add SSH key。

右击打开git bash测试链接

输入ssh git@github.com

以上显示成功。

3.创建Github库。

打开Github主页左上角点击NEW

Repository name = （用户名） + github.io

创建完后博客地址为：https://(用户名).github.io

4.1安装hexo

在一个盘里面新建文件夹来存放hexo程序文件。打开文件夹，右击打开git bash here。输入npm install -g hexo-cli安装很慢而且可能会错误，耐心等待。（正常显示因该是added package而不是update
